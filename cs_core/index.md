# CS Core Courses

These are the foundational courses for the Computer Science major at CMU.

| Course | Name |
|--------|------|
| [15-122](/cs_core/15122) | Principles of Imperative Computation |
| [15-150](/cs_core/15150) | Principles of Functional Programming |
| [15-210](/cs_core/15210) | Parallel and Sequential Data Structures and Algorithms |
| [15-251](/cs_core/15251) | Great Ideas in Theoretical Computer Science |
| [15-451](/cs_core/15451) | Algorithm Design and Analysis |

::: tip Note
15-213 (Introduction to Computer Systems) is cross-listed as 18-213. See the [ECE Core](/ece_core/) section for that course.
:::

## Recommended Sequence

1. **15-122** - After 15-112, often taken with 21-127 (Concepts)
2. **15-150** - After 15-122
3. **15-213/18-213** - After 15-122
4. **15-210** - After 15-150 and 15-122
5. **15-251** - After 15-150 and 21-127
6. **15-451** - After 15-210 and 15-251
